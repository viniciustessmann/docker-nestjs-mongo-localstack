# Docker NestJS + MongoDB + localstack


## comandos
Rodar projeto
`docker-compose up -d`

Accessar projeto API Gerenciadora:
https://localhost:3000

Acessar o banco de dados mongo
http://admin:admin123@localhost:8081

Check localstack
https://localhost:4566/health
